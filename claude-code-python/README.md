# Claude Code Python (CCP)

> 🐍 Python 实现的 Claude Code CLI - 终端 AI 编程助手

⚠️ **项目状态：** 设计阶段 | **预计完成：** 21 天

---

## 🎯 项目目标

用 Python 重写 Claude Code CLI，提供：
- 终端交互式 AI 编程体验
- 文件读写/编辑能力
- 安全的命令执行
- 代码搜索与理解
- MCP/LSP 协议支持

---

## 🚀 快速开始

```bash
# 安装
pip install claude-code-python

# 配置
ccp config

# 运行
ccp run "帮我重构这个项目的代码结构"

# 交互模式
ccp
```
