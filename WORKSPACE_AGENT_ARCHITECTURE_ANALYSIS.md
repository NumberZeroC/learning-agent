### 模块设计

| 模块 | 文件路径 | 职责 |
|------|----------|------|
| **主程序** | `src/main.py` | StockAgent 类、配置加载、任务调度 |
| **新闻监控** | `src/news_monitor.py` | 财经新闻抓取、板块情绪分析 |
| **资金流分析** | `src/capital_flow.py` | 板块主力资金流入分析 |
| **股票筛选** | `src/stock_selector.py` | 基于新闻+资金流推荐股票 |
| **股票监控** | `src/stock_monitor.py` | 个股重点监控（日 K/分时/资金博弈） |
| **报告生成** | `src/report_generator.py` | Markdown/JSON报告生成 |
| **数据源** | `src/tushare_pro_source.py` | Tushare Pro 数据获取（600 积分） |
| **数据源** | `src/akshare_source.py` | AKShare 免费数据获取 |
| **数据源** | `src/enhanced_data_source.py` | ETF/北向资金/宏观数据 |

### 定时任务

| 任务 | 时间 | 脚本 | 功能 |
|------|------|------|------|
| 🌙 晚间分析 | 20:00 | `scheduled_evening.sh` | 大盘走势、新闻情绪、资金流 |
| 📈 早盘推荐 | 9:30 | `scheduled_morning.sh` | 5 个热点板块 + 各 3 只龙头股 |
| 🎯 交易监控 | 30 分钟/次 | `scheduled_monitor.sh` | 交易时间实时监控持仓 |
| 🧹 日志清理 | 08:00 | `cleanup_logs.sh` | 清理旧报告和日志 |
| 📬 通知发送 | 按需 | `send_notify.sh` | QQ 推送报告通知 |

### 数据流

```
数据获取 → 新闻分析 → 资金流分析 → 选股 → 报告生成 → QQ 推送
   ↓           ↓           ↓          ↓         ↓         ↓
Tushare    板块情绪    净流入     龙头股    MD/JSON   qqbot
AKShare    利好/利空   排名       推荐      报告      推送
```

### 技术特点
- **多数据源冗余：** Tushare Pro + AKShare + 网页抓取
- **缓存机制：** 数据缓存 TTL 300-600 秒
- **重试机制：** 最大重试 3 次
- **报告保留：** 14 天在线 + 10 个压缩包归档
