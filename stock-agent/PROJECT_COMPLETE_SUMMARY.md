## 🚀 使用流程

### 1. 环境准备

```bash
cd /home/admin/.openclaw/workspace/stock-agent
source venv_ak/bin/activate
```

### 2. 自动选股

```bash
python auto_trade.py --mode screen --strategy balanced --top 10
```

**输出：**
- 推荐股票 TOP10
- 综合评分
- 各维度得分
- JSON 结果文件

---

### 3. 自动交易

```bash
# 使用选股结果执行交易
python auto_trade.py --mode trade -r data/selection_*.json
```

**执行：**
- 风控检查
- 仓位计算
- 订单执行
- 交易记录

---

### 4. 实时监控

```bash
python auto_trade.py --mode monitor --interval 60
```

**功能：**
- 每 60 秒检查
- 自动止损
- 自动止盈
- 账户统计

## 📋 配置文件

### config.yaml（关键配置）

```yaml
# 账户配置
account:
  initial_capital: 1000000  # 100 万

# 数据源
tushare:
  token: "your_token"
  priority: 1

akshare:
  enabled: true
  priority: 2

# 选股
stock_selection:
  default_strategy: 'balanced'
  recommendations:
    max_count: 10

# 风控
risk_limits:
  max_single_position: 0.30
  stop_loss_pct: 0.08
  take_profit_pct: 0.20
```
