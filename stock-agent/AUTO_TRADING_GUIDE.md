
## 📊 选股策略

### 策略 1：平衡策略 (balanced)

**权重：** 基本面 30% + 技术面 35% + 资金流 25% + 情绪 10%

**适合：** 大多数市场情况

```bash
python auto_trade.py --strategy balanced
```

---

### 策略 2：价值策略 (value)

**权重：** 基本面 40% + 技术面 30% + 资金流 20% + 情绪 10%

**选股条件：**
- PE < 25
- ROE > 15%
- 营收增长 > 20%

**适合：** 中长线投资

```bash
python auto_trade.py --strategy value
```

---

### 策略 3：动量策略 (momentum)

**权重：** 基本面 20% + 技术面 45% + 资金流 25% + 情绪 10%

**选股条件：**
- 20 日涨幅 > 10%
- 突破 60 日新高
- 量比 > 2.0

**适合：** 短线交易

```bash
python auto_trade.py --strategy momentum
```

---

### 策略 4：资金流策略 (capital)

**权重：** 基本面 25% + 技术面 25% + 资金流 40% + 情绪 10%

**选股条件：**
- 主力净流入 > 2000 万
- 机构净买入 > 1000 万

**适合：** 跟随主力

```bash
python auto_trade.py --strategy capital
```

## ⚙️ 配置说明

### config.yaml

```yaml
# ============================================
# 选股配置
# ============================================
stock_selection:
  # 默认策略
  default_strategy: 'balanced'
  
  # 选股范围
  stock_pool:
    exclude_st: true          # 排除 ST
    min_market_cap: 50e8      # 最小市值 50 亿
  
  # 基本面阈值
  fundamental:
    pe_max: 30
    pb_max: 5
    roe_min: 10
    revenue_growth_min: 10
  
  # 技术面阈值
  technical:
    volume_ratio_min: 1.5
  
  # 资金流阈值
  capital:
    main_force_net_min: 5000000  # 500 万
  
  # 推荐数量
  recommendations:
    max_count: 10
    min_score: 60

# ============================================
# 风控配置
# ============================================
risk_limits:
  # 仓位限制
  max_single_position: 0.30    # 单只股票最大 30%
  max_sector_exposure: 0.50    # 单板块最大 50%
  min_cash_ratio: 0.10         # 最低现金 10%
  
  # 止损止盈
  stop_loss_pct: 0.08          # 8% 止损
  take_profit_pct: 0.20        # 20% 止盈
  
  # 交易限制
  max_daily_trades: 10         # 每日最多 10 笔
```

## 🎯 交易规则

### 买入规则

| 规则 | 说明 |
|------|------|
| **单只上限** | 不超过总资产 10% |
| **现金保留** | 至少保留 10% 现金 |
| **100 股整数倍** | 最小交易单位 |
| **风控检查** | 通过才能买入 |

---

### 卖出规则

| 规则 | 说明 |
|------|------|
| **止损** | 亏损>8% 全部卖出 |
| **止盈** | 盈利>20% 卖出一半 |
| **优先级** | 止损 > 止盈 > 新买入 |

---

### 风控规则

| 规则 | 限制 |
|------|------|
| **单只股票** | ≤ 30% 总资产 |
| **单板块** | ≤ 50% 总资产 |
| **现金比例** | ≥ 10% 总资产 |
| **日交易次数** | ≤ 10 笔/日 |

## 📊 选股因子

### 基本面（30 分）

| 因子 | 权重 | 阈值 |
|------|------|------|
| PE 市盈率 | 8 分 | < 30 |
| PB 市净率 | 5 分 | < 5 |
| ROE 净资产收益率 | 8 分 | > 10% |
| 营收增长率 | 5 分 | > 10% |
| 非 ST | 4 分 | 必须 |

---

### 技术面（35 分）

| 因子 | 权重 | 说明 |
|------|------|------|
| MACD | 12 分 | 金叉加分 |
| 均线排列 | 10 分 | 多头加分 |
| 量比 | 8 分 | > 1.5 |
| 突破形态 | 5 分 | 突破加分 |

---

### 资金流（25 分）

| 因子 | 权重 | 阈值 |
|------|------|------|
| 主力净流入 | 12 分 | > 500 万 |
| 机构净买入 | 8 分 | > 0 |
| 融资净买入 | 5 分 | > 0 |

---

### 情绪面（10 分）

| 因子 | 权重 | 说明 |
|------|------|------|
| 新闻情绪 | 5 分 | 正面加分 |
| 板块热度 | 3 分 | 热门加分 |
| 龙虎榜 | 2 分 | 上榜加分 |
