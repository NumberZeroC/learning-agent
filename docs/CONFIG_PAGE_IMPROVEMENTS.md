### 2. **API Key 显示优化** ✅

**修改前：**
```
输入框显示：sk-sp-1103f012953e45d984ab8fbd486e6e16
提示文字：阿里云 DashScope 大模型密钥
```

**修改后：**
```

提示文字：大模型 API 密钥，支持阿里云、OpenAI、DeepSeek 等
下方显示：当前显示：sk-sp11***6e16 | 支持平台：阿里云 DashScope、OpenAI GPT、DeepSeek、智谱 AI 等
```

**美化规则：**
- 显示前 8 位 + `***` + 后 4 位
- 例如：`sk-sp1103f012953e45d984ab8fbd486e6e16` → `sk-sp11***6e16`
- 未配置时显示：`未配置`

## 🛠️ 技术实现

### 新增函数

```javascript
// 美化 API Key 显示（隐藏中间部分）
function maskApiKey(apiKey) {
    if (!apiKey || apiKey.length < 16) return '未配置';
    // 格式：sk-sp11***3434（显示前 8 位和后 4 位）
    const prefix = apiKey.substring(0, 8);
    const suffix = apiKey.substring(apiKey.length - 4);
    return `${prefix}***${suffix}`;
}
```
