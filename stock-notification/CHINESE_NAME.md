## 📖 名字寓意

> **股灵通** - 股票消息灵通，先知先觉

- **股** - 股票、股市
- **灵** - 灵敏、智能、灵通
- **通** - 通达、通知、推送

**整体寓意：** 像股市中的"千里眼"和"顺风耳"，消息灵通，先人一步发现机会！

```
股灵通 (Stock-Notification)
```

## 📝 使用方式

### 口头称呼
- "股灵通" - 日常称呼
- "小灵通" - 亲切叫法（可选）

### 命令行
```bash
# 项目路径
cd /home/admin/.openclaw/workspace/stock-notification

# 执行脚本
./venv311/bin/python3 evening_analysis.py  # 晚间总结
./venv311/bin/python3 morning_recommend.py  # 早盘推荐
./venv311/bin/python3 technical_analysis.py # 技术选股
```

- `FEATURES_SUMMARY.md` - 功能说明
- `CRON_SETUP.md` - 定时任务配置
