# 通知系统更新日志

## 2026-04-02 - 通知内容优化

### 修改内容

**修改前：**
- 通知消息展示报告文件路径
- 用户需要点击链接查看报告

**修改后：**
- 通知消息直接展示报告内容摘要
- 不再显示报告文件路径
- 用户可以直接在 QQ 中看到关键信息

### 修改文件

`notify_report.py`

### 修改详情

1. **read_report_summary 函数**
   - 新增 `content` 字段用于存储报告内容
   - 从 JSON 数据文件中提取报告要点
   - 从 MD 文件中提取前几段内容作为摘要

2. **send_notification 函数**
   - 移除报告文件路径显示
   - 添加报告内容展示
   - 精简消息格式

### 消息格式对比

**修改前：**
```
🌙 *晚间市场总结* 已完成

📌 上证：3948.55 (+1.46%)

要点：
• 热点板块：创新药、半导体、医药生物

📄 报告：`/path/to/report.md`

_投资有风险，决策需谨慎_
```

**修改后：**
```
🌙 *晚间市场总结*

📌 上证：3948.55 (+1.46%)

要点：
• 热点板块：创新药、半导体、医药生物

晚间市场整体表现强势，三大指数全线上涨...

_投资有风险，决策需谨慎_
```

### 用户需求

- 用户希望直接在 QQ 通知中看到报告内容
- 不需要展示报告文件路径
- 内容长度限制在 500 字以内
