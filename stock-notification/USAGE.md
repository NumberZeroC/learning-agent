## 核心功能

### 🌙 晚间分析 (evening_analysis.py)

**功能：**
- 大盘走势分析（5 大指数）
- 全天新闻收集（5 大财经媒体）
- 板块资金流分析
- 新闻情绪分析
- 重点关注板块推荐
- 明日策略建议

**输出：**
```
data/reports/evening_summary_YYYY-MM-DD.md
data/reports/evening_summary_YYYY-MM-DD.json
```

**执行：**
```bash
./venv311/bin/python evening_analysis.py
```

---

### 📈 早盘推荐 (morning_recommend.py)

**功能：**
- 加载晚间报告数据
- 获取早间新闻
- 分析集合竞价（交易时间）
- 推荐 5 个热点板块
- 每个板块推荐 3 只龙头股

**推荐逻辑：**
| 因素 | 权重 |
|------|------|
| 晚间资金流 | 40% |
| 新闻情绪 | 30% |
| 早间新闻 | 30% |

**输出：**
```
data/reports/morning_recommend_YYYY-MM-DD.md
data/reports/morning_recommend_YYYY-MM-DD.json
```

**执行：**
```bash
./venv311/bin/python morning_recommend.py
```

---

### 🎯 交易监控 (scheduled_monitor.sh)

**功能：**
- 监控持仓股票
- 实时资金流分析
- 买卖信号识别
- 生成监控报告

**执行时间：** 交易时间每 30 分钟

**输出：**
```
data/reports/monitor/stock_monitor_YYYYMMDD_HHMMSS.md
```

## 配置说明

### config.yaml

```yaml
# ============ 数据源配置 ============
akshare:
  cache_dir: data/cache
  cache_ttl: 300        # 缓存时间（秒）
  max_retries: 3        # 最大重试次数

tushare:
  token: your_token     # Tushare Pro Token
  cache_ttl: 600

# ============ 分析板块 ============
sectors:
  - 半导体
  - 人工智能
  - 新能源
  - 医药生物
  - 消费电子
  - 汽车
  - 化工
  - 券商
  - 银行
  - 军工

# ============ 新闻关键词 ============
news_keywords:
  positive:
    - 利好
    - 突破
    - 增长
    - 上涨
    - 创新高
  negative:
    - 利空
    - 下跌
    - 风险
    - 亏损
    - 监管

# ============ 资金流阈值 ============
capital_flow:
  threshold_inflow: 5000    # 主力流入阈值（万元）
  threshold_outflow: 5000   # 主力流出阈值（万元）

# ============ 输出目录 ============
output:
  report_dir: data/reports
  log_dir: logs
```

## 数据源

### 主要数据源

| 数据 | 主数据源 | 备用数据源 |
|------|----------|------------|
| 大盘指数 | Tushare Pro | AKShare |
| 个股行情 | AKShare | 新浪财经 |
| 板块资金流 | AKShare | 本地缓存 |
| 财经新闻 | 同花顺/金融界 | 证券时报/上证报 |

### Tushare Pro

**优势：** 稳定、数据质量高
**要求：** 需要积分（600 分以上）
**获取：** https://tushare.pro

### AKShare

**优势：** 免费、数据丰富
**注意：** 可能不稳定，已配置故障转移

## 故障排查

### 常见问题

**1. 数据获取失败**
```bash
# 检查网络
curl -I https://www.akshare.xyz

# 清理缓存
rm -rf data/cache/*

# 检查 Tushare Token
cat config.yaml | grep token
```

**2. 报告未生成**
```bash
# 查看日志
tail -50 logs/cron_evening.log

# 手动运行测试
./venv311/bin/python evening_analysis.py
```

**3. 定时任务未执行**
```bash
# 检查 crontab
crontab -l

# 检查 cron 服务
systemctl status cron
```

### 日志位置

